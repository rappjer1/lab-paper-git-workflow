# Reviewer Response Round Example

This example shows a generic response-round binder.

```bash
paper-scaffold demo --output scratch/demo_manuscript --overwrite
paper-scaffold reviewer-binder --manuscript-repo scratch/demo_manuscript --round 1 --output scratch/reviewer_response_round_1 --overwrite
```

The binder is a starting point for response tracking. Keep confidential review text out of public repositories unless publication rules allow it.

# Manuscript Repository

This is a clean manuscript repository. Keep research code, raw data, model
outputs, caches, and full evaluation directories in the research repository.

Use this repo for:

- LaTeX manuscript source.
- Publication figures and tables copied from the research repo.
- References.
- Supplement source.
- Metadata manifests that explain where figures and tables came from.

Run:

```bash
paper-scaffold validate
```

before pushing changes to GitHub or syncing with Overleaf.

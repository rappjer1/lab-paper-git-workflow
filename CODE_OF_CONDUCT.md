# Code Of Conduct

Be respectful, constructive, and specific.

This project is for researchers and contributors with different backgrounds and experience levels. Harassment, personal attacks, and discriminatory language are not acceptable.

When giving feedback:

- Focus on the work, not the person.
- Assume good intent, but correct problems clearly.
- Keep discussions relevant to the project.

Maintainers may remove comments, close discussions, or block participants who make the project unsafe or unproductive.
